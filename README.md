Sinopsis Aplikasi
-----------------
Aplikasi ini adalah aplikasi yang akan membantu dan memudahkan seseorang dalam mempelajari Bahasa jerman. Aplikasi yang sertai dengan kuis dan penyebutan vocal ini, membuat suasana belajar Bahasa jerman menjadi lebih menyenangkan dan mengasikan.
Dalam aplikasi ini user akan di berikan beberapa menu yaitu *“Anfanger”*, *“Lernen”*, *“Spilen”*. 
*Anfanger* yang berarti pemula maka pada menu ini si user akan mempelajari dasar-dasar dalam berbahasa jerman.
*Lernen* adalah menu yang di peruntukan bagi user yang memiliki tingkatan menengah dimana dia sudah mengetahui dasar-dasar dalam bahasa jerman. 
*Spilen* adalah kuis yang berguna untuk mengasah daya ingatan si user setelah dia mempelajari Bahasa jerman tersebut. 

Manfaat Aplikasi
----------------
- Memudahkan seseorang dalam mempelajari Bahasa jerman
- Membuat seseorang berminat untuk mempelajari Bahasa jerman
- Membuat suasana belajar bahasa jerman menjadi lebih menyenangkan.

Pengguna

2. Remaja
3. Orang Dewasa
